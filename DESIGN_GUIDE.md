# KaiwuDB 官网配色规范 (White & Blue Theme)

本文档定义了 KaiwuDB 官网原型的新配色系统，采用清爽的**白蓝风格**，旨在传达专业、高效、可靠的品牌形象。

## 1. 核心色板 (Color Palette)

### 主色 (Primary)
- **品牌蓝 (Brand Blue)**: `#0056D2`
  - 用途：主要按钮、选中状态、图标、重要链接
  - 悬停态：`#0042A1` (加深 10% 以增加交互反馈)
  - 浅色背景：`#E6F0FF` (用于悬停背景或装饰性元素)

### 背景色 (Background)
- **主背景 (Primary BG)**: `#FFFFFF` (纯白)
  - 用途：页面主体、卡片背景、导航栏
- **次级背景 (Secondary BG)**: `#F5F7FA` (浅灰)
  - 用途：区分区块（如应用场景）、Hero 渐变起点
- **页脚背景 (Footer BG)**: `#0A192F` (深蓝黑)
  - 用途：页脚区域，保持稳重感

### 文字色 (Typography)
- **主要文字 (Primary Text)**: `#1D1D1F` (深灰)
  - 用途：标题、导航链接、强调文本
  - 对比度：与白色背景对比度为 13.6:1 (符合 WCAG AAA)
- **次要文字 (Secondary Text)**: `#6E6E73` (中灰)
  - 用途：正文、描述性文本
  - 对比度：与白色背景对比度为 5.3:1 (符合 WCAG AA)

## 2. 界面元素应用规范

### 按钮 (Buttons)
- **实心按钮 (Primary)**:
  - 背景：`#0056D2`
  - 文字：`#FFFFFF`
  - 圆角：`4px`
  - 阴影：悬停时出现蓝色投影 `rgba(0, 86, 210, 0.3)`
- **轮廓按钮 (Outline)**:
  - 边框：`#0056D2`
  - 文字：`#0056D2`
  - 背景：透明 -> 悬停变浅蓝 `#E6F0FF`

### 卡片 (Cards)
- **默认状态**:
  - 背景：`#FFFFFF`
  - 边框：`1px solid rgba(0,0,0,0.05)` (极细微边界)
  - 阴影：`0 4px 20px rgba(0, 0, 0, 0.05)` (柔和阴影)
- **悬停状态**:
  - 上浮：`translateY(-10px)`
  - 边框：`#0056D2`
  - 阴影：`0 10px 30px rgba(0, 86, 210, 0.15)` (带蓝色的加深阴影)

### 导航栏 (Navigation)
- **样式**: 白色玻璃拟态 (`rgba(255, 255, 255, 0.9)`)
- **交互**: 滚动时增加底部阴影
- **链接**: 默认深灰，悬停变蓝，底部出现蓝色线条指示

## 3. 无障碍性 (Accessibility)
- 确保所有关键文本与背景的对比度至少为 **4.5:1**。
- 交互元素（如 Focus 状态）应有清晰的视觉轮廓。
- 颜色不作为传达信息的唯一手段（结合图标和文本标签）。
